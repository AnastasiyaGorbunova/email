## Скачивание библиотек для работы приложения

Выполнить в терминале команду:

```
npm install
or
npm ci
```

## Компиляция и горячая перезагрузка браузера для разработки

Запускаем локальный сервер на Node.js на порту "3000"

```
npm start
```
